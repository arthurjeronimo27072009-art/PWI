# 🌐 Atlas

Atlas é uma landing page institucional desenvolvida para simular uma plataforma moderna de cursos de programação. O projeto apresenta serviços, projetos e informações da empresa com foco em experiência do usuário, responsividade e organização de código.

---

# 🚀 Tecnologias Utilizadas

- HTML5
- Tailwind CSS
- JavaScript
- Git
- GitHub

---

# 🎯 Proposta do Projeto

A Atlas simula uma plataforma de ensino de programação para iniciantes, com foco em aprendizado prático e acessível.

O objetivo é aplicar conceitos reais de desenvolvimento front-end, como:
- Estruturação de layouts modernos
- Responsividade completa
- Componentização visual
- Organização de projeto em equipe

---

# 📱 Responsividade

O projeto segue o padrão de breakpoints do Tailwind CSS.

| Tamanho | Prefixo | Largura |
|---------|----------|----------:|
| Mobile | padrão | < 640px |
| Small | `sm` | ≥ 640px |
| Medium | `md` | ≥ 768px |
| Large | `lg` | ≥ 1024px |
| Extra Large | `xl` | ≥ 1280px |
| 2XL | `2xl` | ≥ 1536px |

## 📌 Padrão adotado

- Mobile First
- Layout adaptado progressivamente
- Componentes consistentes entre todas as seções
- Tipografia e espaçamentos padronizados

# 💬 Convenção de Commits

```text
feat: nova funcionalidade

fix: correção

style: alteração visual

refactor: reorganização de código

docs: atualização do README

chore: manutenção
```

Exemplos:

```bash
git commit -m "feat: adiciona navbar responsiva"

git commit -m "fix: corrige menu mobile"

git commit -m "style: melhora animações da navbar"
```
