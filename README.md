
Instruções
Para salvar as alterações:

git clone <url do repositório>

Ex: 

git clone https://github.com/Atlas-1613/atlasweb
Para salvar as alterações:

git add .
Para empacotar as alterações:

git commit -m '<notas>' 
Para subir para o repo remoto (GitHub):

git push 
Para atualizar o repositório local:

git pull
Criar branch:

git chekout -b <nome da branch>
Para salvar as credenciais:

  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"
